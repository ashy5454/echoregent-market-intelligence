This repository contains the **Startup Founder Market Intelligence & NLP Pattern Analysis Engine** evaluating **100,000 AI developer discussions** (scraped across `r/ChatGPTCoding`, `r/LocalLLaMA`, `r/LangChain`, `r/MachineLearning`, `r/OpenAI`, and `r/Artificial`) specifically tailored for **Echoregent** ([https://echoregent-yudi-pub.web.app/](https://echoregent-yudi-pub.web.app/)).

---

## 📂 Repository Contents

* 📄 **[`STARTUP_FOUNDER_ANALYSIS.md`](./STARTUP_FOUNDER_ANALYSIS.md)** — Comprehensive VC & Founder Pitch Strategy Report.
* 📜 **[`echoregent_founder_analysis.py`](./echoregent_founder_analysis.py)** — Python NLP Word Frequency & Opportunity Extractor.
* 🌐 **[`dashboard/index.html`](./dashboard/index.html)** — Interactive Visual Founder Dashboard.
* 📊 **[`reddit_100k_ai_dataset.csv`](./reddit_100k_ai_dataset.csv)** — Complete 100,000 Reddit AI discussions dataset.

```bash
python echoregent_founder_analysis.py
```

### 2. View the Visual Founder Dashboard
Open `dashboard/index.html` in your web browser!
